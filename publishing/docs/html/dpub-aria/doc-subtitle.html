Caution

The doc-subtitle role is often treated as a heading, causing the very problem it was meant to solve. It is no longer recommended.

Now that the HTML outline algorithm has been removed, the hgroup element can be used to group a heading with its subtitle(s).

Refer to the knowledge base page on headings for more information.

Description

An explanatory or alternate title for the work, or a section or component within it.

Relation to ARIA

doc-subtitle inherits its semantics from the a sectionhead role.
