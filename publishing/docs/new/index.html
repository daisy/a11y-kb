This page is continuously updated to reflect significant changes to the DAISY Accessible Publishing Knowledge Base. For a complete list of changes, refer to the project's GitHub commit history.

Changes are listed from most recent to oldest. Updates more than a year old are periodically removed from the list.

July 7, 2020 — Main Content

A new page describing the main element, and why its generally not needed for digital publications, is now available.

June 30, 2020 — Text Spacing

A new page describing WCAG's recommendations for text spacing to improve readability is now available.

June 29, 2020 — Contrast

A new page describing the contrast requirements necessary to meet WCAG requirements is now available. Ensuring sufficient contrast between foreground text and background colors improves the overall readability of publications.

June 19, 2020 — Content Lists

A new page about including content lists (lists of tables, illustrations, etc.) is now available.

June 17, 2020 — Site Search

The ability to search the site for topics is now available. A link to the search page is provided at the top of every knowledge base page.

June 16, 2020 — Fixed Layouts

An expanded section on improving the accessibilty of fixed layouts is now available. Although difficult to make fully-accessible, fixed layouts should not be made without accessibility in mind. It is expected that this section will grow and improve as more issues are identified.

May 25, 2020 — EPUB Validation

A new section describing how to validate EPUB files for accessibility is now available. This section covers EPUBCheck, Ace by DAISY and the Ace SMART tools.

May 1, 2020 — Hidden content

A new page detailing various methods for hiding content using HTML, CSS an ARIA is now available. The page also explores the use of scripting and progressive enhancement to hide content in a more accessible manner when it must be hidden.

April 23, 2020 — Improved readability

The default font size and line spacing are now increased to improve the overall readability of the site.

April 20, 2020 — Bibliographies

A new page that details common accessibility issues with bibliographies is now available.

April 17, 2020 — Text to speech

A set of pages covering text-to-speech features available in EPUB has been restored and updated from the old IDPF guidelines. Support for these features remains poor, but this fact is now more prominently noted.

April 17, 2020 — Indexes

A new page that details common accessibility issues with indexes is now available.

April 17, 2020 — Expanded coverage of accessibility metadata

The schema.org accessibility metadata properties are now detailed on their own separate pages. In addition, each page now provides a detailed explanation of the values that the property accepts.

February 28, 2020 — Glossary

A glossary of technical terms used in the knowledge base is now available. A link to the glossary is available at the top of each page. A future update will look at linking terms from where they are used to their definitions.

January 22, 2020 — DPUB-ARIA roles

A new section that details the Digital Publishing WAI-ARIA 1.0 module is now available. Each role is explained in detail on its own page.

January 13, 2020 — ARIA landmarks

A new section that details the proper use of ARIA landmarks is now available.
