This page is continuously updated to reflect significant changes to the DAISY Accessible Publishing Knowledge Base. For a complete list of changes, refer to the project's GitHub commit history.

Changes are listed from most recent to oldest.

October 21, 2020 — Core Media Types

A new page describing the function of core media types in EPUB, when fallbacks are needed, and where resources have to be hosted is now available.

October 20, 2020 — Links to Reference Marks

Linking to where footnotes, figures, tables, etc. are referenced in the text is a useful feature for users, but presents a number of challenges. A new page describing these issues around linking to reference marks is now available.

October 6, 2020 — MathML

The guidance on providing accessible math has been updated to reflect the latest testing from the DIAGRAM Center.

July 23, 2020 — Heading Groups

A new page describing why the hgroup element should not be used to group multiple headings is now available.

July 22, 2020 — Details Element

A new page describing the details element for including additional information in an expandable HTML widget is now available.

July 17, 2020 — Audiobooks Accessibility Metadata

Examples showing how to format the schema.org accessibility metadata for inclusion in an Audiobooks manifest are now available. Additional guidance for audiobooks will be included in future updates.

July 16, 2020 — Expandable Topics

Due to the increasing number of categories and topics covered on the site, the menus on the home page are now expandable. The categories are collapsed by default to simplify their navigation, but each can be individually expanded by clicking on its label or they can all be expanded by selecting the "expand all topics" checkbox at the top of the list.

July 15, 2020 — Expanded ARIA Sections

The scripted content page dealing with ARIA has been broken out into three new pages so that more complete coverage can be provided for the unique topics. The new pages describe custom controls, control labels and the tab order. More feature-specific pages like these will be added in the future.

July 9, 2020 — Reporting Errors and Requesting Topics

Two new links are now provided at the bottom of every page to report errors and request new topics for the knowledge base. To be more user-friendly, instead of going directly to the GitHub repository for the site, these links now lead to a reporting page with a walkthrough of the process.

July 7, 2020 — Main Content

A new page describing the main element, and why its generally not needed for digital publications, is now available.

June 30, 2020 — Text Spacing

A new page describing WCAG's recommendations for text spacing to improve readability is now available.

June 29, 2020 — Contrast

A new page describing the contrast requirements necessary to meet WCAG requirements is now available. Ensuring sufficient contrast between foreground text and background colors improves the overall readability of publications.

June 19, 2020 — Content Lists

A new page about including content lists (lists of tables, illustrations, etc.) is now available.

June 17, 2020 — Site Search

The ability to search the site for topics is now available. A link to the search page is provided at the top of every knowledge base page.

June 16, 2020 — Fixed Layouts

An expanded section on improving the accessibility of fixed layouts is now available. Although difficult to make fully accessible, fixed layouts should not be made without accessibility in mind. It is expected that this section will grow and improve as more issues are identified.
