OCF
OCF Container

OCF stands for Open Container Format, which is the specification that defines the EPUB container format.

OCF, Container, EPUB Container and OCF Container Pare used interchangeably to talk about the ZIP file container for EPUBs.

OEBPS

OEBPS stands for Open EBook Publication Structure, which was the precursor specification to EPUB 2.0.

The acronym OEBPS is sometimes still found used as the directory name where EPUB content is stored in the container but is not recommended.

OPF file

OPS
OPS Content Document

OPS stands for Open Publication Structure, which is the EPUB 2 equivalent specification of the EPUB Content Documents 3 specification.

Similar to OEBPS, OPS is sometimes found as the directory name where EPUB content is stored but is no longer recommended.

Outline Algorithm

The outline algorithm defines a way to create a document outline from the headings in an HTML document. It was an attempt to correct the mess of headings and inconsistent use of numbering that are often found in documents and produce a consistent hierarchy.

The algorithm proved to be confusing for authors, however, and went unimplemented in user agents, so is largely only considered as a theory anymore. User agents will not correct documents, so using proper heading structure in publications is imperative.
