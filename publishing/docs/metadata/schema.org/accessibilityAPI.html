Definition

Indicates that the resource is compatible with the referenced accessibility API.

Examples

Values

This property is generally not applicable to digital publications as the means by which information is conveyed to accessibility APIs is a reading system concern.

Only the following value is recommended for this property:

ARIA

Note

Possible switching of this value to an accessibility feature is currenty under discussion. See issue 14 for more information.

The ARIA value is used to indicate that ARIA roles, states and properties have been used to make scripted content accessible.
