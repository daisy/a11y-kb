Definition

<ProductFormFeatureDescription> contains a short explanatory summary of the accessibility of the product, consistent with the more specific conformance and feature details provided. The summary should note both the accessibility features provided and any potential deficiencies. More detailed information may be provided in an external file using codes 94â€“96.
