概要

バージョン： WCAG 2.0以上
セクション番号： 3.3.5
レベル： AAA
適用対象：
  • 入力フォーム
要件：

コンテキストに応じたヘルプが利用できる。
ガイダンス：
  • 達成基準3.3.5の理解
  • 達成基準3.3.5を満たす方法
達成方法：
  • G71：すべてのウェブページにヘルプへのリンクを提供する
  • G89：期待されるデータ書式及び入力例を提供する
  • G184：フォーム又はテキストフィールド一式の先頭に、必須の入力を記述するテキストの説明を提供する
  • G193：ウェブページでアシスタントによるヘルプを提供する
  • G194：テキスト入力に対するスペルチェック及び修正候補を提供する

目的

この達成基準の目的は、複雑なフィールドの目的と、どのような入力が必要かを説明する指示を提供することで、ユーザーがフォームの入力をスムーズに完了できるようにすることです。

要件の満たし方

For input fields that are not easily understood from their name alone, authors also provide the option for users to access context-sensitive help instructions.

The instructions might provide an example of the expected input or explain in more detail the kinds of information the user is expected to input.
