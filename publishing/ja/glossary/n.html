NCX（エヌシーエックス）

The Navigation Control file for XML Applications (NCX) provides content navigation in the DAISY Digital Talking Book (DTB) and EPUB 2 formats.

NCX ファイルは、両方の形式の目次のソースです。DAISY デジタル録音図書（DTB）形式では、NCX ファイルによって、音声再生順序（オーディオブック用）を定義する機能や、テキストと音声の同期を定義する機能も提供されます。

EPUB 3では、NCX はナビゲーション ドキュメントとメディア オーバーレイの組み合わせに置き換えられました。
