User Agent（ユーザーエージェント）

A user agent is some combination of hardware and/or software used to render the content of a publication (i.e., literally the agent users use to consume content).

リーディングシステムという用語はEPUBでよく使用されますが、リーディングシステムは電子出版物のユーザーエージェントのより具体的な名前にすぎません。

The nature of any user agent is highly dependent on the context in which the term is used (e.g., web specifications typically use the term user agent to refer to browsers).
