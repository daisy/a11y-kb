Package Document（パッケージ ドキュメント）
Package File（パッケージ ファイル）

パッケージドキュメントは、 EPUB出版物の主要な側面を定義するXML文書です。これは、EPUB出版物をレンダリングするためにリーディングシステムによって使用されます。

パッケージ ドキュメントは、出版物に関するメタデータ（タイトル、著者、発行者など）、すべてのリソースのリスト（マニフェスト）、およびコンテンツ ドキュメントを表示する順序（スパイン） という3つの主要なセクションで構成されます。

Page List（ページリスト）

The page list is a navigation aid that provides access to static page break locations (i.e., to locations that do no change depending on the font and screen size).

The page list is particularly useful in mixed print-digital environments (e.g., in a school where some students may need an accessible digital version of a textbook), as it allows everyone to find the same location regardless of the format used.

通常、ページ区切りは、印刷版またはその他の静的にフォーマットされた出版物のバージョンに対応しますが、出版社は、場所の調整を容易にするために（つまり、ユーザーが使用するリーディングシステムに関係なく同じ場所にアクセスできるようにするため）、純粋にデジタルの作品にページ区切りを追加することもあります。
